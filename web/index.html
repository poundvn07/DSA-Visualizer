About Project & Core Architecture

Mini Project Report & System Design

DSA Visualizer is a highly decoupled educational platform designed to bridge the gap between abstract algorithmic logic and interactive visual feedback.

Core Architectural Highlights:

  • Python Generator-Based Simulation: The backend core leverages Python's yield generators to lazily evaluate and capture discrete algorithmic states (comparisons, swaps, variable updates). This approach ensures true "step-by-step" execution without complex threading or blocking operations.
  • Complete Decoupling (Eel Bridge): The heavy lifting of the algorithm logic and benchmarking is fully isolated in the Python backend. The frontend (HTML/JS/CSS) is completely agnostic to the algorithmic implementation and only acts as a dumb renderer subscribing to the backend's state events via the Eel WebSockets bridge.
  • High-Performance Event Logging: By serializing state changes into lightweight JSON "Steps", the system can instantly record an entire sorting/searching process of a large dataset and playback locally without constant client-server roundtrips, eliminating visual lag.
  • Stateless DOM Manipulation: The UI dynamically reconstructs visual components directly from the incoming data snapshots (Array states, active indices), making features like "Rewind" or "Theme Switching" reliable and robust.

Help & Tutorial

Algorithm Visualizer & Benchmarking

  1. Select algorithm type (Sorting / Searching)
  2. Choose an algorithm from the dropdown
  3. Adjust array size and speed with sliders
  4. Pick input type (Random / Sorted / Reversed)
  5. Click New to generate a fresh array
  6. Use Play for continuous animation, or Step buttons for manual control
  7. ⏪ Step goes backward through cached steps
  8. Reset returns to the initial array
  9. Click a bar to edit its value (idle mode only)
  10. Enter comma-separated values in Custom Array and click Apply
